# Vendor Onboarding Process

## Step 1 – Architecture Review

Vendor must submit:

- Network topology
- Data flows
- Firewall requirements
- Remote access method

## Step 2 – Standards Compliance

Must comply with:

- Ignition standards
- MQTT topic structure
- Naming conventions
- Security zoning

## Step 3 – Cybersecurity Review

- No inbound OT connections
- MFA required for remote access
- Jump host or VPN only

## Step 4 – FAT Validation

Validate:

- Tag naming
- Alarm structure
- Data buffering

## Step 5 – SAT Approval

No production access until all issues closed.
