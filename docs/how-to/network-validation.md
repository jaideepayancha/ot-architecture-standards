# Network Validation Checklist

## Before Commissioning

- [ ] VLANs match standard
- [ ] Firewalls installed and tested
- [ ] Redundancy validated
- [ ] No flat networks detected

## Security

- [ ] No inbound IT routes
- [ ] MFA enabled for remote access
- [ ] Logging enabled

## Performance

- [ ] Latency < 10 ms PLC to HMI
- [ ] No packet loss under load
- [ ] Storm control enabled
