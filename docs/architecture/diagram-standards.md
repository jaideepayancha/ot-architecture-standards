# Diagram Standards

## Tools

- Diagrams must be created using **draw.io (diagrams.net)**
- Editable files stored in: `/diagrams/source`
- Exported PNG/SVG stored in: `/diagrams/export`

## Naming Convention

Format:

plantname_layer_purpose.drawio

Examples:

- danville_network_overview.drawio
- bg_scada_architecture.drawio

## Color Standards

| Layer | Color |
|--------|--------|
| PLC / Field | Blue |
| Edge | Green |
| Control | Yellow |
| Servers | Red |
| Enterprise | Purple |
| Cloud | Light Blue |

## Required Diagrams

Every project must include:

- Network topology
- Purdue model alignment
- Data flow
- Remote access
