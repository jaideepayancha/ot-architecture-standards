# Zones and Conduits (IEC-62443)

## Zones

| Zone | Examples |
|--------|--------|
| Cell/Area | PLCs, drives |
| Control | HMIs, SCADA |
| Site Ops | Historians, MES |
| Industrial DMZ | Brokers, jump hosts |
| IT | ERP, AD |

## Conduits

Conduits must:

- Be firewall-controlled
- Have documented rules
- Use least privilege

## Risk Classification

Each zone rated:

- Low
- Medium
- High

Controls scale with risk level.
