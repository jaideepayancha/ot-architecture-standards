# OT Cybersecurity Governance

## Objectives

- Protect personnel, production, and equipment
- Prevent cyber incidents impacting operations
- Maintain regulatory and insurance compliance

## Standards Alignment

- IEC 62443 series
- NIST SP 800-82
- ISA-99

## Scope

Applies to:

- PLCs
- HMIs
- SCADA servers
- Edge gateways
- OT network infrastructure

## Roles

| Role | Responsibility |
|--------|--------------|
| OT Architect | Architecture standards |
| IT Security | Policy enforcement |
| Plant Engineering | System maintenance |
| Vendors | Compliance with standards |
