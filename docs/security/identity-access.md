# Identity and Access Management

## Authentication

- Central AD integration where possible
- Local accounts only if isolated

## Authorization

Role-based access:

| Role | Example Access |
|--------|-------------|
| Operator | View + acknowledge alarms |
| Engineer | Tag edits |
| Admin | Gateway config |

## MFA

Required for:

- VPN
- Jump hosts
- Cloud services

## Account Reviews

Quarterly review of:

- Vendor accounts
- Contractor access
