# Backup and Disaster Recovery

## Backup Scope

- PLC programs
- HMI projects
- SCADA gateways
- Network configs

## Frequency

| System | Backup |
|--------|-------|
| PLC | After changes |
| SCADA | Daily |
| Network | After config changes |

## Storage

- Offline copy required
- Immutable storage preferred

## Recovery Testing

- Annual restore tests
- Document recovery time
