# OT Incident Response

## Objectives

- Protect safety first
- Restore operations quickly
- Preserve evidence

## Incident Types

- Malware detection
- Network intrusion
- Unauthorized access

## First Actions

1. Isolate affected zone
2. Maintain control system stability
3. Notify OT + IT security

## Forensics

- No re-imaging before evidence capture
- Preserve logs and configs

## Recovery

- Rebuild from known-good backups
- Validate before reconnecting
