# HMI and Dashboard Design Standards

## Layout

- Follow corporate UI standards
- Neutral background colors
- High contrast for alarms

## Navigation

- No more than 3 clicks to reach equipment
- Breadcrumb navigation required

## Templates

- All equipment uses templates
- No copy-paste graphics

## Performance

- No SQL queries in bindings
- Use tag-driven views

## Mobile

- Perspective views must support tablet resolution
